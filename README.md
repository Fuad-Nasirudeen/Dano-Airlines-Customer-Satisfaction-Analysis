# Dano-Airlines-Customer-Satisfaction-Analysis
Dano Airlines experienced a drop in customer satisfaction below 50% for the first time. This project analyzes passenger survey data to identify key drivers of dissatisfaction and provide data-driven recommendations to improve overall customer experience.

![](https://github.com/Fuad-Nasirudeen/Dano-Airlines-Customer-Satisfaction-Analysis/blob/main/images(1).jpeg)

## Objective
-	Identify factors influencing passenger satisfaction
-	Compare satisfaction across travel types and customer segments
-	Recommend actionable strategies to improve satisfaction rate

## Dataset
-	Source: Airline passenger satisfaction survey
-	Records: 120,000+ passengers
-	Scope: Passenger demographics, flight details, service ratings
- Key variables:
  
  -	Satisfaction
  
  -	Type of travel
  
  -	Customer class
  
  -	Service ratings (cleanliness, comfort, inflight service, delays)
  
  -	Arrival and Departure Delays
 
## Data Preparation
- Removed duplicates
-	Handled missing values
-	Standardized categorical fields
-	Converted satisfaction into binary classification (Satisfied / Not Satisfied)


## Skills/ Concepts Demonstrated
- Data Cleaning and Transformation
- Data Visualization
- Quick measures
- Dashboard Automation
